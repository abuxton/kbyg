# Puppets Module Skeleton #
## Your out the box starter for 10! ##

When you run `puppet module generate name-module` you get given the following.


    @@@ SH
        a-module/
        ├── README.md
        ├── Rakefile
        ├── manifests
        │   └── init.pp
        ├── metadata.json
        ├── spec
        │   ├── classes
        │   │   └── init_spec.rb
        │   └── spec_helper.rb
        └── tests
            └── init.pp

##We are not sure this is enough! But its more than you used to get.
